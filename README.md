# SimulationProject
Проект "Симуляция" выводит в консоль поле  M на N с рандомно расположенными сущностями.
Хищные поедают травоядных, травоядные едят траву. За ход теряется N количество здоровья.
Когда останется одно животное игра закончится.

Сущности:
Predator - хищник
Herbivore - травоядное
Grass -  трава
Tree - дерево
Rock - камень

Настройки находятся в resources:
fieldRowCol.properties - размеры поля
icons.properties - цвета сущностей
settingEntity.properties - параметры сущностей

За расчет пути отвечает FindPathAlgorithm, выдается ближайший путь до еды животного, занятые ячейки не дают прохода, поедая занимает клетку.
Если нет прохода до еды, животное остается на месте теряя здоровье каждый ход. Расчет происходит для каждого хода.

Возможно запустить полный цикл пока все животные не вымрут, или пошагово(это оди цикл хода всеми животными)
